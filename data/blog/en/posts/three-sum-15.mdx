## Problem

Given an integer array `nums`, determine whether there exist triples `[nums[i], nums[j], nums[k]]` such that `i != j`, `i != k`, and `j != k`, and `nums[i] + nums[j] + nums[k] == 0`. Return all unique triples that sum to 0.

**Note**: The answer must not contain duplicate triples.

**Example**

> Input: `nums = [-1, 0, 1, 2, -1, -4]`
> Output: `[[-1, -1, 2], [-1, 0, 1]]`

## Approach

**Prerequisite**: you can implement the two-sum variant with two pointers on a sorted array.

1. Because indices are not required in the output, sort `nums` and iterate. After sorting, e.g. `newNums`: [-4, -1, -1, 0, 1, 2].
2. 3Sum can be viewed as: fix a `head` index, then run two pointers for a simplified “two sum” with `target = 0 - newNums[head]`.
   ![threeSum](https://raw.githubusercontent.com/yanchengfang/note-assets/refs/heads/master/algorithm/threeSum.png "threeSum")
   > It is “simplified” because a **strictly increasing** array removes many edge cases.
3. Outer loop over `head` (the first element of the subproblem). Two rules:
   - If `head`’s value is ≥ 0, stop: in a sorted array, no pair to the right can make the sum ≤ 0 with a non-negative first value.
   - Skip duplicates for `head`; otherwise you may emit duplicate triples or waste work.

   ```js
   var threeSum = function (nums) {
     let newNums = nums.sort((a, b) => a - b);
     let res = [];
     for (let i = 0; i < newNums.length - 2; i++) {
       if (nums[i] > 0) break;
       if (i !== 0 && nums[i] === nums[i - 1]) {
         continue;
       }
       let head = nums[i];
       let l = i + 1; // left pointer
       let r = newNums.length - 1; // right pointer

       //...
     }
   };
   ```

4. Move `l` and `r` until `l >= r`, then advance `head`.

   ```js
   while (l < r) {
     let resultNum = head + nums[l] + nums[r]; // compare sum of three positions with 0
     if (resultNum === 0) {
       res.push([head, nums[l], nums[r]]);
       // if we need non-duplicate triples, only moving both pointers can find another hit
       r--;
       l++;
       continue;
     } else if (resultNum > 0) {
       // sum too large: need smaller values, move r left
       r--;
     } else {
       // sum too small: need larger values, move l right
       l++;
     }
   }
   ```

5. To avoid duplicates, skip equal neighbors on both pointers.

   ```js
   while (l < r) {
     if (l !== i + 1 && nums[l] === nums[l - 1]) {
       // skip duplicate l (not the first of the window)
       l++;
       continue;
     }
     if (r !== newNums.length - 1 && nums[r] === nums[r + 1]) {
       // skip duplicate r (not the last index)
       r--;
       continue;
     }
     let resultNum = head + nums[l] + nums[r];
     // ...
   }
   ```

6. When `l` and `r` meet, move `head` to the next distinct value.

## Full code

```js
var threeSum = function (nums) {
  let newNums = nums.sort((a, b) => a - b);
  let res = [];
  for (let i = 0; i < newNums.length - 2; i++) {
    if (nums[i] > 0) break;
    if (i !== 0 && nums[i] === nums[i - 1]) {
      continue;
    }
    let head = nums[i];
    let l = i + 1;
    let r = newNums.length - 1;
    while (l < r) {
      if (l !== i + 1 && nums[l] === nums[l - 1]) {
        l++;
        continue;
      }
      if (r !== newNums.length - 1 && nums[r] === nums[r + 1]) {
        r--;
        continue;
      }
      let resultNum = head + nums[l] + nums[r];
      if (resultNum === 0) {
        res.push([head, nums[l], nums[r]]);
        r--;
        l++;
        continue;
      } else if (resultNum > 0) {
        r--;
      } else {
        l++;
      }
    }
  }
  return res;
};

let nums = [-1, 0, 1, 2, -1, -4];
let nums2 = [-2, 0, 3, -1, 4, 0, 3, 4, 1, 1, 1, -3, -5, 4, 0];
console.log(threeSum(nums2));
```

### Complexity

> - Time: O(n^2)
> - Space: O(1)
